########Português

Esse projeto foi criado para fins de estudo. Todo conhecimento abordado aqui foi utilizado a fim de
exercitar PHP, MVC, HTML e Javascript.

O projeto consiste em exibir um guia de TV, mostrando os próximos 10 programas que serão exibidos com hora de início e fim.
O programa que estiver em exibição está destacado.
A cada mudança de hora a página é automaticamente atualizada para exibir o guia atual.
Somente um administrador logado pode acessar a programação completa bem como fazer inclusões, alterações ou exclusões.
Durante a inserção/alteração cada campo é válidado em nível de back-end e front-end.
O guia completo é exibido em páginas e cada página corresponde a um dia da semana.

Antes de rodar o projeto é necessário carregar o script no banco de dados MySQL.
Fique à vontade para alterar o guia de acordo com os programas que voce achar melhor.
Altere o usuário e senha na classe de conexão de acordo com as configurações do seu banco.
Para iniciar o servidor do PHP basta usar o comando 'php -S localhost:8080' na pasta public


########English

This project was created for study purpose. All the knowledge covered here was used in order to exercise PHP, MVC, HTML and Javascript.

The project consists of displaying a TV guide, showing the next 10 TV shows that will be shown with a start and end time.
The show that is currently running is highlighted.
With each change of time the page is automatically updated to display the current guide.
Only a logged in administrator can access the complete programmation as well as make additions, changes or deletions.
During insertion/change each field is validated at the back-end and front-end level.
The complete guide is displayed in pages, and each page corresponds to a day of the week.

Before running the project it is necessary to load the script into the MySQL database.
Feel free to change the guide according to the programs you think best.
Change the user and password in the Connection class according to your database settings.
To start the PHP server just use the command 'php -S localhost:8080' in the public folder.
